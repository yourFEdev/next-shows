A modern and responsive portfolio website built with **Next.js**, **TypeScript**, and **Tailwind CSS** to showcase my skills, experience, and featured projects as a Frontend Web Developer.

![Next.js](https://img.shields.io/badge/Next.js-15-black?logo=nextdotjs)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?logo=typescript)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-4-38BDF8?logo=tailwindcss)

## ✨ Features

-  Dark / Light Mode
-  Fully Responsive Design
-  Modern UI with Tailwind CSS
-  Smooth Animations using Framer Motion
-  Download Resume
-  Featured Projects Showcase

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- next-themes

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourFEdev/next-shows.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
